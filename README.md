# 🌍 Smart World Clock & Time Zone Comparator

A C++ console-based application that displays the current time of multiple countries, compares time zones, and suggests suitable meeting times between different regions.

## 📌 Features

- Display current local time of multiple countries
- Support for 10 countries
- Search current time of a specific country
- Compare time differences between two countries
- Suggest common meeting hours based on working hours (9 AM – 5 PM)
- Display time in both 24-hour and 12-hour formats
- Menu-driven interface
- Input validation for better user experience
